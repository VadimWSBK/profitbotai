I have connected Shopify to my ProfitBot MCP server. The ProfitBot MCP server now has access to Shopify tools that allow me to manage orders, customers, and products in my Shopify store.

**Shopify Integration Status:**
- Shopify is connected to my ProfitBot workspace
- The MCP server has access to Shopify Admin API through the connected store
- All Shopify operations are scoped to my workspace's Shopify store

**Available Shopify Tools:**

### Order Management
- `shopify_list_orders` - List recent orders (limit: 1-250, default: 10)
- `shopify_search_orders` - Search orders by query (query: required, limit: 1-50, default: 10)
- `shopify_get_order` - Get detailed order info (orderId: required, numeric)
- `shopify_cancel_order` - Cancel an order (orderId: required, optional: reason, notify, restock)
- `shopify_refund_order` - Refund an order in full (orderId: required, optional: notify, note)
- `shopify_get_customer_orders` - Get all orders for a customer (customerId: required, numeric)

### Customer Management
- `shopify_list_customers` - List customers (limit: 1-250, default: 50, optional: pageInfo)
- `shopify_get_customer` - Get customer details (customerId: required, numeric)

### Products & Analytics
- `shopify_list_products` - List products (limit: 1-250, default: 100)
- `shopify_get_statistics` - Get store statistics (days: 1-365, default: 30)

**How to Use:**

When users ask about Shopify orders, customers, or store info:
- **Check order status**: Use `shopify_search_orders` or `shopify_get_order`
- **Cancel order**: Use `shopify_cancel_order` with order ID
- **Refund order**: Use `shopify_refund_order` with order ID
- **Find customer orders**: Use `shopify_get_customer_orders` with customer ID
- **Look up customer**: Use `shopify_search_orders` by email or `shopify_list_customers`
- **Get store stats**: Use `shopify_get_statistics`

- When canceling/refunding, you can notify the customer and restock inventory
